# 网络即兴剧场（Qt 6 / C++ / TCP）

## 功能
- TCP 服务端 + Qt Widgets 客户端
- 演员最多 5 人，观众最多 10 人
- 中文 UI
- 服务端控制台命令：
  - `1`：开始表演，随机分配场景和演员角色
  - `2`：结束表演并开启观众投票
  - `3`：结束投票并公布最佳演员
  - `q`：关闭服务端
- 演员可发送文字消息，观众可观看
- 语音采用“麦克风开/关”模式：
  - 演员点击麦克风按钮开始采集并发送 PCM 音频
  - 再次点击停止
  - 其他客户端实时播放
- 观众投票时只能投给演员

## 环境
- Qt 6.x
- Qt Creator
- qmake
- 模块：core、network、widgets、multimedia

## 构建
分别打开：
- `Server/Server.pro`
- `Client/Client.pro`

先运行 Server，再运行多个 Client。

## 测试建议
1. 启动服务端。
2. 启动 2 个演员客户端和 1 个观众客户端。
3. 服务端输入 `1`。
4. 演员进行文字或语音交流。
5. 服务端输入 `2`。
6. 观众投票。
7. 服务端输入 `3`。
