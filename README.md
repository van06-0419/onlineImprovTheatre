# onlineImprovTheatre
Сетевое импровизационное театральное приложение с ролями, чатом и голосованием зрителей.

## Описание проекта

**Online Improv Theatre** — это клиент-серверное приложение, разработанное на языке **C++** в рамках курсового проекта.

Приложение предназначено для проведения онлайн-импровизационного театра. Пользователи делятся на две группы:
​- 1.**Актёры** (до 5 человек), которые получают случайные роли и участвуют в импровизации.
​- 2.**Зрители** (до 10 человек), которые наблюдают за выступлением и по его завершении голосуют за лучшего актёра.
Общение между участниками осуществляется через сетевой чат. После окончания представления система подсчитывает результаты голосования и определяет победителя.

## Основные возможности
- подключение клиентов к серверу;
- управление комнатой;
- распределение ролей между актёрами;
- проведение импровизации;
- чат между участниками;
- голосование зрителей;
- подсчёт результатов голосования.
